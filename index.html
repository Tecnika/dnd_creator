🤖 Как генерируются карточки

Карточки создаются вручную через интерфейс или загружаются в формате JSON. Каждая карточка относится к определённому типу (персонаж, враг, оружие, локация и т.д.) и содержит набор полей, специфичных для этого типа.

Генерация AI: В будущем карточки смогут генерироваться с помощью AI на основе вашего описания. Пока что все карточки создаются вручную.

🏷️ Общие карточки

Карточки с меткой "общее" доступны во всех играх всем пользователям. Это удобно для универсальных предметов, навыков или NPC, которые можно переиспользовать.

Чтобы сделать карточку общей, отметьте флаг "Общее" при создании или редактировании.

🌳 Структура сюжетного древа (storyTree)

Сюжетное древо хранится в документе игры и создаётся через визуальный редактор. Пример структуры:

📦 Полный экспорт игры (JSON)

При экспорте игры через кнопку 💾 Экспорт сохраняется полная структура:

{
  "game": {
    "name": "Гнилой оброк",
    "setting": "Древняя Русь",
    "description": "Детектив в славянском сеттинге."
  },
  "rulesMode": "simplified",
  "storyTree": {
    "nodes": [ /* … */ ],
    "edges": [ /* … */ ]
  },
  "cards": [
    { "type": "player", "data": { /* … */ }, "isCommon": false },
    { "type": "enemy", "data": { /* … */ }, "isCommon": false }
  ]
}

⚙️ Как это работает

Приложение использует Firebase Firestore для хранения данных и Firebase Authentication для входа.

  • После регистрации вы создаёте игры
  • В каждой игре можно создавать карточки разных типов
  • Карточки можно редактировать, удалять, экспортировать
  • Карточки с меткой "общее" видны всем пользователям
